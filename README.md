# 🦎 Geeko Swaying Animation

A smooth **SVG animation of a climbing gecko with swaying leaves and blinking eyes**, built using **HTML, CSS, and SVG animations**.  
The animation creates a natural swaying effect to simulate movement on a branch.

---

## ✨ Features

- Pure **HTML + CSS animation**
- Smooth **SVG-based graphics**
- **Blinking eye animation**
- **Leaf sway effect**
- **Branch swinging motion**
- Lightweight and responsive

---

## 🛠️ Built With

- HTML5
- CSS3
- SVG Animation
